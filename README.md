# ytWeb

Web app to download YouTube videos up to 1080p with audio.

## Prerequisites
- Node.js 18+
- `ffmpeg` available in system PATH (required to merge video+audio)

## Setup
```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Notes
- This app accepts YouTube URLs (`youtube.com` or `youtu.be`).
- Only download content you own or have permission to download.
